loadstring delivery Junkie SDK Roblox UI
01 — Overview

What is JunkieHub?

JunkieHub is a Roblox key system UI library. Loaded externally via loadstring and configured entirely from your own script — no modifying the library. When a valid key is redeemed, it sets getgenv().LOAD = true and your game script continues.

TIP — The library is CDN-hosted and loaded remotely. You only write a loader script with your config — stefan handles the library internals.
02 — Flow

How It Works

1
Load the Library
Your script calls loadstring(game:HttpGet(LIBRARY_URL))() — fetches the library from CDN and returns a Hub object.
2
Configure & Init
Pass your Config to Hub.new(Config) then call hub:Init() to render the key system UI.
3
Wait for Redemption
while not getgenv().LOAD do task.wait(1) end blocks until a valid key is entered, then execution continues.
03 — Template

Minimal Template

Lua
local system = loadstring(game:HttpGet("LIBRARY_URL"))()
local Config = { -- your config here }
local hub = system.new(Config)
hub:Init()
while not getgenv().LOAD do
  task.wait(1)
end
-- game script runs here
04 — Core

Junkie_Configs

Controls Junkie SDK integration — key validation, webhook relay, and provider setup.

PropertyTypeDescription
UseJunkiebooleantrue loads the Junkie SDK for real-time key validation. false accepts any key (testing mode).
Junkie_WebhookstringYour Junkie relay webhook URL. Notifies you on key redemption. Leave "" to disable.
ServicestringYour service name from the Junkie dashboard. Must match exactly.
IdentifierstringYour user/dashboard identifier from Junkie.
ProviderstringThe provider name configured in your Junkie dashboard.
MaxKeyAttemptsnumberFailed attempts before UI force-closes. Recommended: 5.
Premiumbooleantrue loads the Premium script URL from GameSupport instead of Free.

Providers

Array of key provider buttons shown in the UI. Each can be shown, hidden, or pinned.

PropertyTypeDescription
NamestringDisplay name on the provider button.
Iconstringrbxassetid for the icon. Use "rbxassetid://0" for the built-in default.
UrlstringURL opened on click. Fallback if get_key_link() returns nil.
Showbooleantrue = visible. false = hidden.
IsJunkiebooleanExactly ONE provider should have this. Enables SDK's get_key_link() flow.
Pinnedbooleantrue = rendered first with a highlight border.
WARNING — Only one provider should have IsJunkie = true. If none, all providers render as regular URL buttons.

GameSupport

Maps Roblox Place IDs to script URLs. After key acceptance the matching script loads. Universal is required as a fallback.

Lua
GameSupport = {
  [2788229376] = {
    Name    = "Da Hood",
    Free    = "https://api.jnkie.com/.../DAHOOD_FREE_HASH/download",
    Premium = "https://api.jnkie.com/.../DAHOOD_PREMIUM_HASH/download",
  },
  Universal = {
    Name    = "Universal",
    Free    = "https://api.jnkie.com/.../UNIVERSAL_FREE_HASH/download",
    Premium = "https://api.jnkie.com/.../UNIVERSAL_PREMIUM_HASH/download",
  },
}
WARNING — Replace all HASH placeholders with your actual Junkie script hashes. The Universal entry is required.
05 — Visual

UI_Configs

Controls the visual appearance and behaviour of the key system UI.

General

PropertyTypeDescription
NamestringHub display name in the title section.
VersionstringVersion pill under the hub name. e.g. "1.0"
ColorColor3Primary accent colour — buttons, borders, glows, badges. e.g. Color3.fromRGB(138, 43, 226)
Text_ColorColor3 / ""Text on coloured buttons. Leave "" to auto-pick black/white from Color brightness.
Iconstringrbxassetid for your hub logo in the title bar.
BadgeTextstringBadge label next to the icon. e.g. "KEY SYSTEM"
CustomFontstringrbxassetid of a custom font. Leave "" for built-in Gotham.
ShowOutlinebooleantrue = accent-coloured border around the main panel.

Background & Overlay

PropertyTypeDescription
GradientAnimatedbooleantrue = background gradient slowly rotates.
GradientAnimationSpeednumberSeconds per full gradient rotation. Lower = faster.
GlassOpacitynumberPanel transparency. 0 = opaque, 1 = invisible. 0.18 = frosted glass.
SlideInEnabledbooleantrue = panel slides up on open. false = instant appear.
BlurEnabledbooleantrue = applies BlurEffect to Lighting while UI is open.
BlurSizenumberBlur strength 0–56. Higher = more blur.
OverlayOpacitynumberBackground overlay darkness. 0 = none, 1 = black. 0.65 = dimmed.

Snow

PropertyTypeDescription
SnowEnabledbooleantrue = falling snowflake particles over the background.
SnowDensitynumberSnowflake count multiplier. 0.5 = half, 2.0 = double. Default: 1.0
SnowColorColor3Snowflake colour. Default: Color3.fromRGB(220, 235, 255)
SnowSize{Min, Max}Pixel diameter range. e.g. {Min=3, Max=7}

Key System

PropertyTypeDescription
KeySystemLabelstringLabel text above the key input box.
KeySystemPlaceholderstringAnimated placeholder that types itself when the box is empty.
KeyFilestringFilename to save/load the verified key locally. Change per-hub to isolate keys.
CloseOnSuccessbooleantrue = UI auto-closes after successful redeem.
SuccessMessagestringToast shown on success. e.g. "✓ Access Granted!"

Discord

PropertyTypeDescription
DiscordstringYour invite code (part after discord.gg/). Powers the Join button and live member count.
ShowMemberCountbooleantrue = fetches and shows live Discord member + online count.

Socials & AppLinks

Socials appear in the side panel and footer strip. AppLinks are pill-shaped chips below the socials grid. Both share the same structure.

PropertyTypeDescription
NamestringDisplay name. Also used to auto-resolve icon if Icon is "rbxassetid://0".
Iconstringrbxassetid for the icon image.
LinkstringURL copied to clipboard on click.
Showbooleantrue = visible.
Pinnedbooleantrue = rendered first with a highlight border.
AUTO-ICONS — Set Icon to "rbxassetid://0" for built-in icons: Discord, Twitter, YouTube, TikTok, Instagram, Telegram, Reddit, GitHub, Guilded, Patreon, Ko-fi, Twitch, V3rm, Website, Linkvertise, LootLabs, Work.ink, Shrtfly, Lockr, Cuty, ShrinkEarn, Rinku.

Instructions

Step cards on the How-To page (opened via the ? button). Each card has a title and body description.

Lua
Instructions = {
  { Title = "Open Junkie",      Body = "Click Junkie Development to open your key link." },
  { Title = "Complete Steps",   Body = "Finish all checkpoints on the Junkie page." },
  { Title = "Paste & Redeem",   Body = "Paste your jnk- key above and hit Redeem." },
}

Changelog

Shown when users click the ★ button. Controls tag pill colours and bullet points per version.

PropertyTypeDescription
Showbooleantrue = shows the ★ changelog button.
TitlestringHeader text on the changelog page.
VersionsarrayArray of version entries. Each has Version, Date, Tag, Notes.
Entry FieldTypeDescription
VersionstringVersion pill label. e.g. "1.2"
DatestringOptional date shown right-aligned. e.g. "Feb 2026"
TagstringOne tag per version. Controls pill colour. See tag table below.
Notesstring[]Each string becomes a bullet point on that version card.

07 — Example

Full Config Example

A complete, working config you can copy and customise.

Lua — Full Config
getgenv().LOAD = nil

local Junkie_Configs = {
  UseJunkie       = true,
  Junkie_Webhook  = "https://jnkie.com/api/v1/webhooks/relay/YOUR_ID/send",
  Service         = "My Hub",
  Identifier      = "1234",
  Provider        = "My Hub",
  MaxKeyAttempts  = 5,
  Premium         = false,
  Providers = {
    { Name="Junkie Development", Icon="rbxassetid://0", Url="https://jnkie.com/junkiehub", Show=true, IsJunkie=true, Pinned=true },
    { Name="Linkvertise", Icon="rbxassetid://0", Url="https://linkvertise.com/yourhub", Show=true, Pinned=false },
  },
  GameSupport = {
    Universal = { Name="Universal", Free="https://api.jnkie.com/.../HASH/download", Premium="https://api.jnkie.com/.../HASH/download" },
  },
}

local UI_Configs = {
  Name = "My Hub", Version = "1.0",
  Color = Color3.fromRGB(138, 43, 226), Text_Color = "",
  GradientAnimated = true, GradientAnimationSpeed = 4,
  Icon = "rbxassetid://YOUR_ICON_ID", BadgeText = "KEY SYSTEM",
  CustomFont = "", ShowOutline = true,
  KeySystemLabel = "Enter Key", KeySystemPlaceholder = "jnk-XXXXXXXXXXXXXXXX",
  ShowMemberCount = true,
  SnowEnabled = true, SnowDensity = 1.0,
  SnowColor = Color3.fromRGB(220, 235, 255), SnowSize = { Min=3, Max=7 },
  GlassOpacity = 0.18, SlideInEnabled = true,
  BlurEnabled = true, BlurSize = 30, OverlayOpacity = 0.65,
  CloseOnSuccess = true, SuccessMessage = "✓ Access Granted!",
  KeyFile = "myhub_key.txt", Discord = "your_invite_code",
  Socials = {
    { Name="Discord", Icon="rbxassetid://0", Link="https://discord.gg/your", Show=true, Pinned=true },
    { Name="YouTube", Icon="rbxassetid://0", Link="https://youtube.com/@you", Show=true, Pinned=false },
  },
  AppLinks = {
    { Name="Discord App", Icon="rbxassetid://0", Link="https://discord.gg/your", Show=true, Pinned=true },
  },
  Instructions = {
    { Title="Open Junkie",     Body="Click the Junkie button to open your key link." },
    { Title="Complete Steps",  Body="Finish all checkpoints to get your key." },
    { Title="Paste & Redeem",  Body="Paste your key above and hit Redeem." },
  },
  Changelog = {
    Show=true, Title="What's New",
    Versions = {
      { Version="1.0", Date="Feb 2026", Tag="NEW", Notes={ "Initial release." } },
    },
  },
}
