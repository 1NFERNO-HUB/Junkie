Overview

What is this library and how it works.

Overview

What is it?

This is a Roblox key system UI library. Loaded externally via loadstring and configured entirely from your own script — no modifying the library. When a valid key is redeemed, it sets getgenv().LOAD = true and your game script continues.

TIP — The library is CDN-hosted and loaded remotely. You only write a loader script with your config.

How It Works

The flow of the key system.

Flow

1
Load
Your script calls loadstring(game:HttpGet(URL))() — fetches library from CDN.
2
Configure
Pass your Config to Hub.new(Config) then call hub:Init().
3
Wait
while not getgenv().LOAD do task.wait(1) end blocks until valid key entered.

Minimal Template

Copy and use this basic template.

Template

Lua
local system = loadstring(game:HttpGet("LIBRARY_URL"))()
local Config = { -- config here }
local hub = system.new(Config)
hub:Init()
while not getgenv().LOAD do
  task.wait(1)
end

Full Example

Complete configuration example.

Complete Config

Lua
getgenv().LOAD = nil

local Config = {
  UseJunkie = true,
  Service = "My Hub",
  Name = "My Hub",
  Color = Color3.fromRGB(138, 43, 226),
  SnowEnabled = true,
}

local system = loadstring(game:HttpGet("URL"))()
local hub = system.new(Config)
hub:Init()
while not getgenv().LOAD do
  task.wait(1)
end
